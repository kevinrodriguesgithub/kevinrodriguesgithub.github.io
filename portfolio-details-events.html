Project information

  • Name: EventShare
  • Technologies: React, Node.js, ExpressJS, MongoDB

Description

EventShare is an application which gives details regarding the events happening in Northeastern University. The list of events are displayed and users can click on any individual event to get more details about it. Users can also click on get directions button which will give them the map directions to react at that location for the event.
